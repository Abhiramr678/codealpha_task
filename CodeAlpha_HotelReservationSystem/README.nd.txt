# 🏨 Hotel Reservation System (Java)

A simple console-based Hotel Reservation System developed using Java. This project allows users to view available rooms, book rooms, cancel bookings, and exit the application through a menu-driven interface.

## 📌 Features

- View available and booked rooms
- Book a room
- Cancel a booking
- Simple menu-driven interface
- Uses Object-Oriented Programming (OOP)
- Easy to understand and beginner-friendly

## 🛠 Technologies Used

- Java
- Scanner Class
- Arrays
- Object-Oriented Programming (Classes & Objects)

## 📂 Project Structure

```
HotelReservationSystem.java
```

## 🚀 How to Run

### Step 1: Compile

```bash
javac HotelReservationSystem.java
```

### Step 2: Run

```bash
java HotelReservationSystem
```
