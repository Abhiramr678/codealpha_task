# 🎓 Student Grade Tracker (Java)

A simple console-based Student Grade Tracker developed using Java. This application allows users to enter student names and marks, then calculates and displays the average, highest, and lowest marks along with a student report.

## 📌 Features

* Add multiple students
* Store student names and marks
* Calculate average marks
* Find highest marks
* Find lowest marks
* Display a complete student report
* Simple menu-free console application

## 🛠 Technologies Used

* Java
* Scanner Class
* Arrays
* Loops
* Conditional Statements
* Basic Object-Oriented Programming Concepts

## 📂 Project Structure

```text
StudentGradeTracker.java
```

## 🚀 How to Run

### Step 1: Compile

```bash
javac StudentGradeTracker.java
```

### Step 2: Run

```bash
java StudentGradeTracker
```
