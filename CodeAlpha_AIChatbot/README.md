# 🤖 AI Chatbot - CodeAlpha Java Internship

A simple console-based AI Chatbot developed in Java as part of the **CodeAlpha Java Programming Internship**. The chatbot interacts with users through the command line by responding to predefined questions, displaying the current date and time, and ending the conversation when the user types `exit`.

---

## 📌 Project Overview

The AI Chatbot is a beginner-friendly Java project designed to demonstrate fundamental programming concepts such as user input, conditional statements, loops, and date/time handling. It simulates a basic conversation with users through the console.

---

## ✨ Features

* Interactive console-based chatbot
* Greets the user at startup
* Responds to predefined questions
* Displays the current date and time
* Handles unknown inputs gracefully
* Exits when the user types `exit`

---

## ▶️ Getting Started

### Prerequisites

* Java JDK 8 or later
* VS Code (or any Java IDE)

### Clone the Repository

```bash
git clone https://github.com/your-username/AIChatbot.git
cd AIChatbot
```

### Compile the Program

```bash
javac AIChatbot.java
```

### Run the Program

```bash
java AIChatbot
```
