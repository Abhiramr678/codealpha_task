# 📈 Stock Trading Platform (Java)

A simple console-based Stock Trading Platform developed using Java. This project allows users to view stock information, buy shares, sell shares, and manage their portfolio through a menu-driven interface.

## 📌 Features

* View available stock details
* Buy shares
* Sell shares
* View portfolio
* Track account balance
* Beginner-friendly console application

## 🛠 Technologies Used

* Java
* Scanner Class
* Variables & Operators
* Loops
* Switch Case
* Object-Oriented Programming Basics

## 📂 Project Structure

```
StockTradingPlatform.java
```

## 🚀 How to Run

### Step 1: Compile

```bash
javac StockTradingPlatform.java
```

### Step 2: Run

```bash
java StockTradingPlatform
```
